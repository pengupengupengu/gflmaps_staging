Instructions:
[Select Map]： Choose Campaign and Mission, and also Layer and Zone(s) for applicable maps.

[Download Map]: Downloads the current map view in 4800x2700.
[Reset Map]: Resets map to original scale.
[Hide Map]: Hides map display.
[Toggle Legend]: Shows a legend of various symbols on the map.

[Enemies]: When enabled, shows enemy leader and CE on their initial nodes.
[Buildings]: When enabled, shows buildings on nodes.
[Node Type]: When enabled, shows node type (i.e. HQ, heliport, etc.) symbols on nodes.
[Node Color]: When enabled, shows the initial alignment of each node.
[Locations]: When enabled, shows the spot ID of each node.
[AI]: When this and "Highlight Nodes" are enabled, shows the patrol route and alert range of enemies.

[Buildings Table]: When enabled, if there are buildings on the current map, it will show a list of all buildings. If "Highlight Nodes" is enabled, then clicking on a table row will highlight that building.
[Portals Table]: When enabled, if there are portals on the current map, it will show a list of all portals. If "Highlight Nodes" is enabled, then clicking on a table row will highlight that portal.
[Highlight Nodes]: When enabled, you can also manually enter node locations (spot IDs) into the four inputs and click "Redraw" to highlight those nodes.
[Combine Identical Teams]: When enabled, identical teams on a map will be combined in the Enemies Table

[Team ID]: You can enter enemy team IDs to manually check teams that are not on the current map.
